# Devcamper

Express CRUD API for Devcamper application, which is a bootcamp directory website.

## Run the app locally

### Install dependencies

```bash
npm install
```

### Add environment variables

```bash
cp .env.example .env
```

> Update the environment variables in .env file

### Run the application

```bash
npm run dev
```

## Seed db

```bash
# Import data
npm run data:import
```

```bash
# Destroy data
npm run data:destroy
```
